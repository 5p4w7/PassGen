# Password Generator

This project is a simple Python password generator that creates random passwords with a few built-in safety rules.

## What it does

The script asks the user for a desired password length and then generates a random password that:
- starts with a letter
- excludes the dollar sign ($)
- excludes commas (,)
- excludes quotation marks (")
- excludes spaces

## Purpose

The purpose of this program is to generate passwords that are easy to use and avoid common characters that may cause issues in some contexts.

## How to use it

1. Make sure Python is installed on your computer.
2. Open a terminal or command prompt in the project folder.
3. Run the script with:

   ```bash
   python new-password.py
   ```

4. Then Enter the desired password length when prompted.
5. The program will display a generated password.

## Example

```text
Enter the desired password length: 12
Generated password: A7mQx2pL9kT
```

## Notes

You can modify the script if you want to change the password rules or add more character restrictions.
